== Description ==

Under by indot is a simple plugin to make your scheduled launch or scheduled maintenance task easier! You have access to a countdown where you set the time to launch or finish your maintenance, its easy to use and allow you to work your SEO so even when your site isnt live yet you can get some rank in google!

**Current Features**


* Display Under
* Translate Everything
* Set a Countdown
* Go Live when your Countdown Reaches 0
* Customize your Favicon
* SEO Friendly
* Cool Design
* Whitelist by IP and/or Roles
* Status Code

== Installation ==

Download and install Under by Indot using the built in Wordpress plugin installer. To install the plugin manually, simply upload it to "/wp-content/plugins/" folder.

To start using the plugin activate it in the "Plugins" admin panel by clicking on the "Activate" link.

== Frequently Asked Questions ==

= Can I change every text label? =

Yes you can, everything can be changed except for the colors.

= What about the labels in the count down (days, hours, minutes, seconds) ? =

You have a few languages there, more will be added in the future.

= Will my webpage go live once the count down reaches 0? =

It's up to you, just check the option in your settings page.

= Can I display only the Logo or only the Title? =

Yes you can, you can choose to display or hide every single element in the under page.

== Screenshots ==
1. **Under Screen** - This is how the Under Looks!
2. **Control Panel** - Here is where you activate the plugin and set a title, a content title and a simple message if you wish.
3. **Countdown** - Here you set the estimated time your site will be down or the remaining time to launch it!! If you want, your website will go live on the desired date and time without having to do anything!
4. **About** - Simple page content about your future site or the reason your site is under maintenance.
5. **Footer** - Simply set your footer content.
6. **Whitelist** - Here you can set which people are allowed to see your website, either by the role they have on your website or by their ip!
7. **Status Code** - You can set the http response code, so robots know whats happening on your website!
8. **Seo** - Allows you to customize your seo by adding some important meta tags! Start promoting your website even before you have launched it.
